Play Videos On Demand with Multiple Lines

== Description ==

**Features**
Setup a Videojs player with multiple network lines settings, users can choose their faverate line and play the video.
You can customize the Video player theme, lines and so on

== Installation ==

1. Upload to the wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Visit the plugin settings to configure display options
